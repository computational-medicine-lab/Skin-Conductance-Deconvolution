------------------- Deconvolution ----------------------------
For deconvolution run "deconvolution_on_UT_Dallas_Data.m"

Results will be saved in folder called "deconvolution_results"

Run "deconvolution_results\plot_results.m" to plot the figures

Please copy the "deconvolution_results" folder to the "Stress_State_Estimation_Code" project folder and follow "read_me.txt" file there in order to perform stress estimation.
